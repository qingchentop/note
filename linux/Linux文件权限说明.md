

实例查看：执行命令 ll

```shell
ll
-rw-r--r--. 1 root root 594033 5月   4 17:04 1.jpg
-rw-r--r--. 1 root root    226 5月   6 22:24 afile.txt
drwxr-xr-x. 2 root root      6 5月   8 21:51 test

```

举例说明

```shell
drwxr-xr-x. 2 root root      6 5月   8 21:51 test
#文件所有者root对文件目录(d)具有rwx(读写执行)的权限;
#文件所有者root所在的组root对test目录具有r-x(读和执行)的权限
#文件所有者root所在组以外的所有用户具有r-x(读和执行)的权限

#具体各个字段说明如下：
drwxr-xr-x
#第一个字符表示文件类型：“-”代表普通文件，“d”代表目录
#剩下每三个为一组，每组由rwx组成，如果是“-”表名没权限
#“r”读权限，“w”写权限，“x”执行权限。如果位置是“-”则没有此类权限。

1/2/……
#第二列数字表示文件数量：普通文件为1，目录为2(..与.);

root root
#第一个root是文件所有者
#第二个root是文件所有者所在组


```

```shell
#linux用户：
#1.所有者(u)
#2.所属组(g)
#3.其他用户(o)
#4.所有用户(a)
```
